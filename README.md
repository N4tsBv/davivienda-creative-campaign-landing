Estructura del proyecto

El proyecto está organizado para facilitar la navegación y revisión técnica.

1. Landing principal

/Index.html

 Funciona como página de presentación y punto de acceso a todas las piezas desarrolladas.

Desde aquí se puede navegar hacia:
* Landing 
* Email HTML 
* Piezas gráficas
* Respuestas de liderazgo

2. Maquetación

/Maquetacion/

Contiene el desarrollo técnico de las piezas:

   * /landing/Landing-Dav.html → Landing correspondiente al brief

   * /email/Mail-Dav.html → Maquetación del correo HTML

   * /assets/ → Recursos utilizados (imágenes, SVG y video)

Los estilos y scripts están integrados dentro de cada archivo HTML.


3. Visuales


/visuales/
Incluye los recursos gráficos exportados en formato JPG como entregables visuales.

3. Cómo revisar
      1. Abrir index.html en el navegador.

      2. Navegar desde allí hacia las demás piezas.

      3. Alternativamente, se puede acceder directamente a cada archivo dentro de su respectiva carpeta.
